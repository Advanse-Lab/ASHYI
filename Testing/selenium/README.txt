Para ejecutar bien estos tests, hacer lo siguiente:

- Instalar firefox y el plugin Selenium IDE http://docs.seleniumhq.org/download/
- Copiar un archivo llamado 'hola.txt' a /tmp (el contenido del archivo puede ser cualquier cosa)
- Abrir AshiyEduTestSuite con Selenium IDE

- Antes de ejecutar el test suite, cargar la base de datos con los datos por default que se encuentran en ASHYI/ashiy.sql 
- Ejecutar el test suite
